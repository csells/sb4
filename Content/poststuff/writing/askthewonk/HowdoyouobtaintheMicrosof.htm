Q. How do you obtain the Microsoft VB6 to VB.NET conversion tool? None of the local software suppliers even carry the VB.NET Professional or Enterprise edition software!

Asked by reader. Answered by the Wonk on November 5, 2002

A. The Visual Basic Upgrade Wizard comes with Visual Studio .NET. You can trigger it by choosing File->Open->Project from within VS.NET and choosing the .vbp file that represents your VB6 project. Be warned that the wizard can only do so much and that you’re going to have to finish most migrations manually using the 101-Keyboard Wizard and the 2-Button Mouse Wizard that comes with your computer. Check out the _UpgradeReport.htm file that gets generated by the Upgrade Wizard for tips.
