Q. How do I get my start page back? I need to set up my profile. But when I launch VS, I go right to the design page. I never saw a start page.

Asked by ace9. Answered by the Wonk on December 8, 2002

A. The View->Web Browser->Home menu item will show the Start page. If you’d like to it automatically show whenever you first start VS.NET, you can set Tools->Options->Environment->General->At startup to “Show Start Page” (no quotes). I personally like to set that option to “Show empty environment” (no quotes) because VS.NET starts like the wind if it doesn’t have to host Internet Explorer as it starts up.
